## 可以被重载的运算符
```cpp
+       -       *       /       %       ^
&       |       ~       !       ,       =
<       >       <=      >=      ++      --
<<      >>      ==      !=      &&      ||
+=      -=      /=      %=      ^=      &=
|=      *=      <<==    >>==    []      ()
->      ->*     new     new[]   delete  delete[] 
```
## 不能被重载的运算符
```cpp
::      *       .       ? :
```
